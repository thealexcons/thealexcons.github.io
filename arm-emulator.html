ARM11 Emulator, Assembler and Debugger

The goal of this project was to create an emulator in C that simulates the execution of a program on an ARM11 processor, and an assembler for the ARM11 instruction set that translates assembly code into a binary executable, which can then be run on the emulator itself.

The project was coded in C by a team of four first-year students (including myself) at Imperial College. We also built a GUI debugger for ARM11 binaries which runs within the terminal.
