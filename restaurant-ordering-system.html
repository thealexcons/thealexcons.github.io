Restaurant Ordering System

The goal of this project was to create an ordering system for my dad's local restaurant, which includes both a Just Eat styled web app for customers and a GUI client program for kitchen staff to view the incoming orders.

Technology stack:

  • Web app:
    • Flask web framework (Python), for the web app and backend
    • SQLite3, for the database
    • HTML, CSS and JavaScript + Bootstrap, for the client side
  • GUI program:
    • Windows Forms framework, using C#
